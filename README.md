# Research Grant Allocation Optimizer

A university research funding optimization system using the Fractional Knapsack Algorithm.

## Features

- Greedy Optimization
- Fractional Knapsack
- C++
- Research Grant Allocation
- Time Complexity O(n log n)

## Dataset

10 Research Projects

Budget: ₹50 Lakh

## Algorithm

1. Calculate Impact-to-Cost Ratio
2. Sort Projects
3. Allocate Budget
4. Allow Partial Funding

## Output

Total Research Impact : 752.6
